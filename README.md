## Instalación y Commpilación
``` bash
# Para instalar las dependencias
npm install
# or
yarn install

# Iniciar la aplicación en  localhost:8080
npm run dev
# or
yarn dev

# Compilar la aplicación para producción
npm run build
# or
yarn build
```
